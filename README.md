# Yamaha MT Series Website

A simple HTML & CSS project showcasing Yamaha’s MT motorcycle series.

## 📄 Pages

- `index.html`: Main landing page with hero section and product cards
- `our-bikes.html`: Gallery and info page for MT series bikes

## 🚀 Features

- Responsive layout using CSS Flex and Grid
- Hero section with full-width background image
- Product cards with wishlist (♥) toggle effect
- Font Awesome icons for social media and features
- Embedded YouTube video
- Clean, semantic HTML structure

## 🛠️ Built With

- HTML5
- CSS3
- Font Awesome
